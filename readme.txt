Project Type: Prototype

Suggested Technologies
======================
IDE: Netbeans 12.6 and above, download at: (https://netbeans.apache.org/download/index.html)
Software development environment : JDK 17 and above, download at: (https://www.oracle.com/java/technologies/downloads/)
Server: Glassfish 4.1.1, download at: (https://javaee.github.io/glassfish/download)

Instructions (Required basic web development knowledge)
======================================================
1. Create database table with the sql queries provided at: TechNet_SourceCode/sql/ 
2. There are 2 sql files, "Create Table.sql" and "Insert Record.sql"
3. Open "Create Table.sql" with your favourite text editor
4. "Create Table.sql" consisting 6 database create table SQL queries
5. Run these sql queries in your integrated development environment. (Suggested to run all queries)
6. Open "Insert Record.sql"
7. "Insert Record.sql" consisting data for the database tables
8. Run these sql INSERT queries in your intergrated development environment. (Suggested to run all queries except "INSERT INTO orders" to prevent errors)
9. One can opt out the "INSERT into users" queries and use the sign-up functions provided
10. Create a new project named Technet
11. Import "TechNet_SourceCode/web/" and "TechNet_SourceCode/java/" source codes into your project
